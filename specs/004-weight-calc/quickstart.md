# Quickstart: Автоматический подсчёт веса

## Сценарий 1: Первый ввод 1ПМ

1. Открыть приложение → основная страница программы
2. Нажать кнопку меню (☰) в левом верхнем углу → открывается сайдбар
3. Нажать "1ПМ" → отображаются 3 поля: Жим, Присед, Тяга (все = 0)
4. Ввести: Жим = 100, Присед = 150, Тяга = 200
5. Значения сохраняются автоматически при изменении (debounce)
6. Закрыть сайдбар → на странице программы подходы с процентами теперь показывают вес

## Сценарий 2: Проверка расчёта веса

При 1ПМ жима = 100 кг:
- Подход `70%×4×2` → отображается как `70% (70кг) ×4×2`
- Подход `75%×3×3` → отображается как `75% (75кг) ×3×3`
- Подход `72%×4×2` → отображается как `72% (70кг) ×4×2` (72кг → floor до 70)

При 1ПМ тяги = 200 кг:
- Подход `72%×3` → отображается как `72% (142.5кг) ×3` (144 → floor до 142.5)

## Сценарий 3: Без 1ПМ

- Если 1ПМ жима = 0, подходы жима показывают только процент: `70%×4×2` (без кг)
- ACCESSORY упражнения никогда не показывают расчёт

## Сценарий 4: Изменение 1ПМ

1. Открыть сайдбар → 1ПМ → изменить Жим со 100 на 120
2. Закрыть сайдбар → все подходы жима мгновенно пересчитаны
3. Перезагрузить страницу → значения сохранены, расчёт тот же
